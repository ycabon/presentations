# thumbnail

Object containing a thumbnail image.

**Referenced by:** [slide](slide.md)

### Properties

| Property | Details
| --- | ---
| url | The URI pointing to the thumbnail image. Can be a URL or a base64-encoded image.


### Example

```json
{
  "thumbnail": {
    "url": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAA..."
  }
}
```
